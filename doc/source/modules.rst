Modules
=======

Overview
--------

The project needs from different helpers and auxiliar classes to implement geographical functionality,
exception handling, etc.

The modules are:

- **Geo**: Provides geographical functions to the Metaclass and data models to specific geometries in the database
- **Exceptions**: Provides exception classes to specific errors of the project
- **JSON Decoders**: Provides extra functionality to th ``json.loads`` function
