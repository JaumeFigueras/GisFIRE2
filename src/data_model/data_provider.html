Data Provider

This module defines the SQLAlchemy ORM model for a data provider in the GisFIRE system. A data provider represents an entity that supplies geospatial or lightning data to the platform. It includes metadata such as name, description, and an optional URL, as well as relationships to other models like Lightning.

Notes

The module uses __future__.annotations to allow type annotations referring to the enclosing class (PEP 563).

SQLAlchemy ORM model representing a data provider in the GisFIRE system.

This class stores metadata about data providers, including their name, description, optional URL, and a timestamp of creation through the TimeStamp mixin. It also maintains relationships to associated lightning records.
